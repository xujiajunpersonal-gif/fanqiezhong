# 🍅 番茄钟 (Pomodoro Timer)

简洁的网页版番茄钟，浏览器直接打开即可使用，零依赖。

## ✨ 功能

- **三种模式**：工作 / 短休 / 长休，点击切换
- **自定义时间**：每种模式可自行设置分钟数
- **计时控制**：开始 / 暂停 / 重置
- **重置回到设定时间**：重置后回到你设定的分钟数，而非硬编码默认值
- **圆形进度环**：倒计时同步动画
- **声音提醒**：计时结束播放提示音
- **自动切换**：工作结束自动进入休息，每 4 个番茄后进入长休
- **番茄计数**：当前会话已完成数量

## 🚀 使用

直接用浏览器打开 `index.html` 即可。

### 默认配置

| 模式 | 默认时长 |
|------|---------|
| 工作 | 10 分钟 |
| 短休 | 5 分钟 |
| 长休 | 15 分钟 |

## 🛠 技术栈

- 原生 HTML5 / CSS3 / JavaScript (ES6+)
- 零依赖，无需安装、无需构建
